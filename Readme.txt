Задача.
обучить нейронную сеть lenet на задачу классификации используя dataset cifar10,
 фреймфорк pytorch. Дополнительное условие - подумать над повышение точности, 
возможно изменить архитектура сети, 
обосновать свой выбор. Срок до 23,05
---------------------------------------------------------------------------------------
Используемая литература:

1. https://russianblogs.com/article/6343764825/
2. https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html
---------------------------------------------------------------------------------------
Сысылка на репозиторий:
 https://github.com/IgorVeret/lenets.git
---------------------------------------------------------------------------------------
Используемое ПО:
1. Pycharm PyCharm 2021.3 (Professional Edition) Runtime version: 11.0.13+7-b1751.19 amd64
2. Python 3.10.4 (tags/v3.10.4:9d38120, Mar 23 2022, 23:13:41) [MSC v.1929 64 bit (AMD64)] on win32
3. ОС Windows 10
---------------------------------------------------------------------------------------
Файлы в комплекте
1. 'NetLanet.py' - основной запускаемый файл
2. 'result.pt' - результат обучения сети
3. 'requirements.txt' - использованные библиотеки и их версии
4. 'Вывод консоли.txt' - результат вывода в консоль(файл создан вручную)
---------------------------------------------------------------------------------------
Запуск программы Lenet
1. Создайте новый проект
2. Скопируйте в проект файлы 'NetLanet.py', 'requirements.txt'
3. Командой 'pip install -r requirements.txt' установите необходимые зависимости.
4. Запустите файл 'NetLanet.py'.
5. Дождитесь окончания работы программы. В процессе будут скачаны файлы в папку 'data'
с сайта https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz и распакованы.
6. Будет создан файл result.pt с результатами обучения.
7. Результат в файле 'Вывод консоли.txt'
